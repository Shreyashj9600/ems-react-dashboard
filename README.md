# EMS React Dashboard (Employee Management System)

A modern, responsive, and clean **Employee Management System (EMS) Admin Dashboard** built with **React.js**, **Vite**, and **Tailwind CSS**.

This project provides an admin panel interface to manage employee data, view dashboard statistics, and organize workplace operations efficiently.

---

## Features

 Modern Admin Dashboard UI  
 Sidebar Navigation with Active Links  
 Employee Management Pages  
 Dashboard Overview & Statistics Cards  
 Fully Responsive (Mobile + Tablet + Desktop)  
 Built with Fast Performance using Vite  
 Styled using Tailwind CSS  
 Reusable and Scalable Component Structure  

---

## Tech Stack

- **React.js** (Frontend Library)  
- **Vite** (Fast Build Tool)  
- **Tailwind CSS** (Utility-First Styling)  
- **JavaScript (ES6+)**
