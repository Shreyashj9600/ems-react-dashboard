# EMS React Dashboard

A modern and responsive **Employee Management System (EMS) Dashboard** built using **React.js** and **Tailwind CSS**.  
This project provides a clean admin dashboard UI to manage employees, view statistics, and organize workplace data.

---

## Features

 Modern Dashboard UI  
 Sidebar Navigation  
 Employee Management Pages  
 Responsive Design (Mobile + Desktop)  
 Built with React + Vite  
 Styled using Tailwind CSS  
 Reusable Components Structure  

---

## Tech Stack

- React.js  
- Vite  
- Tailwind CSS  
- JavaScript (ES6+)
